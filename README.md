A Patient Managment System for Medical Practices

Welcome to my final project for Eleven Fifty Academy. This Web Application is future based development to create a well rounded, modern and modular patient
management system. This Web App is currently a work in progress (even at the time of submission) to further implement new ideas, tables and functionalities in
order to make it a flagship style CRM/PMS application.

Current Status: v1.0.1
In this version: Patients, Doctors, and Appointments are live and functionial. All CSS/Bootstrap/Sass/jQuery in functional in the main view home page. 

Bugs/Issues: Currently the Appointment table does not display Doctor names or Patient names properly, working on narrowing down the issue. Patient Profiles and Visits
are not yet available so the links are broken.
